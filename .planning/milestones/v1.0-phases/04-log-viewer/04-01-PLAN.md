**1. Create `server/routes/logs.js`** — new Express router with a single GET `/:service` endpoint:

- Import `execFile` from `node:child_process`, `promisify` from `node:util`, and `SERVICE_NAME_RE` from `../utils/exec.js`.
- Create `execFileAsync = promisify(execFile)`.
- Define `VALID_LINES = /^\d+$/` and `VALID_SINCE = { '5m': '-5m', '15m': '-15m', '1h': '-1h', '6h': '-6h', '1d': '-1d' }` as module-level constants.
- Route handler:
  - Validate `req.params.service` against `SERVICE_NAME_RE`. Return 400 `{ ok: false, error: 'Invalid service name' }` if invalid.
  - Read `req.query.lines` (default `'100'`). Validate with `VALID_LINES` and `parseInt(linesRaw, 10) <= 1000`. Return 400 if invalid.
  - Read `req.query.since` (default `'all'`). Only use `VALID_SINCE[sinceKey]` — any unknown value is treated as 'all' (no `--since` arg).
  - Build args array: `['--no-pager', '-q', '-u', service, '-n', linesRaw, '--output', 'json']`. If `sinceKey !== 'all' && VALID_SINCE[sinceKey]`, push `'--since', VALID_SINCE[sinceKey]`.
  - Call `execFileAsync('/usr/bin/journalctl', args, { timeout: 15_000, maxBuffer: 5 * 1024 * 1024 })`.
  - Parse stdout: `stdout.trim().split('\n').filter(Boolean).map(line => { ... })`.
  - For each JSON line: parse `PRIORITY` (default to 6 if absent), convert `__REALTIME_TIMESTAMP` from microseconds (divide by 1000 for JS Date), guard `MESSAGE` with `Array.isArray` check, extract `SYSLOG_IDENTIFIER` or `_COMM`.
  - Each entry shape: `{ ts: string|null, priority: number, level: 'error'|'warning'|'info', identifier: string, message: string }`.
  - Level mapping: `priority <= 3 ? 'error' : priority === 4 ? 'warning' : 'info'`.
  - Return `{ ok: true, service, entries }`.
  - Handle empty stdout (no entries): return `{ ok: true, service, entries: [] }` — check `stdout.trim() === ''` before splitting.
  - Wrap in try/catch, call `next(err)` on failure.
- Export `default router`.

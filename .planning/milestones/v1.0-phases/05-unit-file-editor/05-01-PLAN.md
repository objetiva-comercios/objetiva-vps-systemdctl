2. Create `server/routes/unit.js` as a new Express router module. Follow the same structure as `server/routes/logs.js` (ESM import, express.Router(), default export).

**Imports needed:**
```javascript
import express from 'express'
import { readFile, writeFile, rename } from 'node:fs/promises'
import { resolve, dirname, basename, join } from 'node:path'
import { randomBytes } from 'node:crypto'
import { SERVICE_NAME_RE, runSystemctl } from '../utils/exec.js'
```

**Constants:**
- `READ_PREFIXES` array: `['/etc/systemd/system/', '/usr/lib/systemd/system/', '/lib/systemd/system/', '/run/systemd/system/']`
- `WRITE_PREFIX` string: `'/etc/systemd/system/'`

**Helper functions:**
- `validatePath(filePath, prefixes)` — calls `resolve(filePath)` to collapse `../` traversal, then checks if the resolved path starts with any of the provided prefixes. Returns boolean.
- `getFragmentPath(service)` — calls `runSystemctl('show', service, ['-p', 'FragmentPath'])`, parses the `FragmentPath=` line from stdout, returns the path string or `null` if empty/not found.

**GET `/:service` endpoint:**
- Validate service name with `SERVICE_NAME_RE.test(service)` — 400 if invalid.
- Call `getFragmentPath(service)` — 404 if null.
- Call `validatePath(fragmentPath, READ_PREFIXES)` — 403 if false.
- `readFile(resolve(fragmentPath), 'utf8')` — catch `ENOENT` and return 404.
- Return `{ ok: true, service, path: fragmentPath, content, writable }` where `writable = resolve(fragmentPath).startsWith(WRITE_PREFIX)`.

**PUT `/:service` endpoint:**
- Validate service name — 400 if invalid.
- Validate `req.body.content` is a non-empty string — 400 if not.
- Guard `content.length > 1_000_000` — 400 if too large.
- Call `getFragmentPath(service)` — 404 if null.
- Check `resolve(fragmentPath).startsWith(WRITE_PREFIX)` — 403 with message "Only files in /etc/systemd/system/ can be edited" if false.
- Atomic write: create temp file path in same directory as destination using `join(dirname(destPath), '.tmp-' + basename(destPath) + '.' + randomBytes(4).toString('hex'))`. Write with `writeFile(tmpPath, content, { encoding: 'utf8', mode: 0o644 })`. Then `rename(tmpPath, destPath)`.
- After successful rename, call `await runSystemctl('daemon-reload', null)`.
- Return `{ ok: true, service, path: fragmentPath }`.
- Wrap in try/catch, pass errors to `next(err)`.
